# Prompt para crear página de bienvenida en PHP + Bootstrap

Crea una **página de bienvenida para una aplicación web de ventas**, desarrollada en **PHP** usando **Bootstrap 5**, con un diseño limpio, profesional y responsive.

IMPORTANTE: Guiarse de la estructura actual del proyecto, solo en el fronentd, ya luego yo me encaragare del backend

el boostrap tiene que estar instalado en css/boostrap

## Requisitos generales

- Utilizar **PHP** como archivo principal (`index.php`).
- Integrar **Bootstrap 5 CDN**.
- Diseño moderno y ordenado.
- Responsive para escritorio y móvil.
- Fondo blanco.
- Barra superior simple con navegación horizontal.

---

## Estructura de la página

### 1. Navbar superior

Crear una barra de navegación clara con:

- Nombre del sistema a la izquierda: **SENATI**
- Menú horizontal con enlaces:
  - Home
  - Productos
  - Marcas

Color de fondo gris claro o blanco con borde inferior suave.

---

### 2. Contenido principal

Debajo del menú colocar un contenedor Bootstrap (`container mt-4`).

Agregar un título grande con icono:

🏛 Aplicación de Ventas

Usar:
- `h1`
- Icono Bootstrap Icons (`bi bi-bank` o similar)

---

### 3. Carrusel principal

Debajo del título agregar un **Bootstrap Carousel** con:

- Altura aproximada: `450px`
- Imágenes de tecnología / laptops / tienda de computadoras
- Bordes suaves
- Flechas laterales
- Transición automática cada 3 segundos

Usar mínimo 3 imágenes.

Ejemplo temática:
- laptops en exhibición
- tienda tecnológica
- computadoras modernas

---

### 4. Estilo visual

Aplicar:

- Márgenes amplios
- Tipografía moderna
- Sombras suaves
- Bordes redondeados en imágenes
- Separación limpia entre elementos

---

## Archivos esperados

- `index.php`
- código organizado
- Bootstrap CDN
- Bootstrap Icons CDN

---

## Código esperado

Genera código completo funcional en PHP + HTML + Bootstrap listo para ejecutar en XAMPP o localhost.
